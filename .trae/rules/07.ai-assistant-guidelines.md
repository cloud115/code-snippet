# JavaScript 代码片段库 - AI助手使用规范

## AI助手使用规范

- **Git提交**：AI助手除非明确指示，否则不得主动执行git commit操作
- **文件修改**：AI助手可以进行文件修改，但修改后应等待用户指示再决定是否提交
- **操作确认**：对于重要操作，AI助手应先向用户确认后再执行
- **代码生成**：AI助手生成的代码应遵循规则文档中的代码风格和规范
- **代码检查**：AI助手生成或修改代码后，必须运行ESLint检查并修复所有错误，确保代码符合规范
  - 运行命令：`npm run lint` 或 `npx eslint <文件路径>`
  - 自动修复：`npx eslint <文件路径> --fix`
  - 特别注意：确保使用Unix风格LF行尾符，而非Windows风格CRLF
